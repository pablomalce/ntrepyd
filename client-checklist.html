How to deliver assets
  1. Create a shared Google Drive or Dropbox folder called NTREPYD — Website Assets
  2. Use subfolders: 01-Logos, 02-Photos, 03-Team, 04-Partners, 05-Copy, 06-Legal
  3. Name files clearly: ntrepyd-logo-primary.svg, maria-garcia-headshot.jpg, etc.
  4. Share the folder link with your designer along with this completed checklist
